选酒：游戏开始后随机选择一位玩家，确定输赢一口气喝几杯酒（一杯酒为一个底分），可以选择1-5杯或者商议自定。

所猜点数从小到大为：2点，3点，4点，5点，6点，1点。

在游戏中点数1既可代表1，也可代表骰钟里任意点数，但是只要有一方喊过1后那么1就只能为1，不能代表其他的点数。

 

游戏介绍：

  1. 每个人摇一次骰子，看清自己骰盅内的点数，并猜测其他玩家的点数。
  2. 随机一名玩家开始叫，猜所有玩家骰盅内共有多少个某点数骰子，叫法为M个N（如3个2点，6个5点，4个4点等）。
  3. 下一名玩家分析判断此叫法是否真实，信之为真则接着叫，叫法同样为M个N，但M（个数）和N（点数）至少有一个要大于上家所叫之数（如上家叫2个5，下家叫2个6，3个4，4个5等均符合规则）。
  4. 若下家不信则开盅验证，统计所有人骰盅内有该点数的骼子个数之和，若的确有不少于M个N点。则上家赢，反之则下家赢（如上家叫5个6，开盅时若只有4个6，则上家输，若有5个或更多个6点，则下家输）。
